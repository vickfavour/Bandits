# 🎭 The Shadow Brotherhood - Bandit Recruitment Site

A fun, interactive recruitment website for a school play about a band of outlaws.

## Overview

This project is a single-page application (SPA) styled as a "wanted poster" themed website 
where visitors can learn about a fictional bandit crew and "apply" to join them. 
It's designed to be humorous, immersive, and easy to present during a school play.

## Features

- **5 Interactive Sections:**
  - 🏠 **The Hideout** - Welcome page with a "WANTED" poster
  - 📜 **Our Legend** - Backstory and famous heists
  - 👥 **The Crew** - Meet the bandit members
  - ⚖️ **The Code** - Rules and initiation trials
  - 🖋️ **Join Up** - Application form with validation

- **Responsive Design** - Works on desktop, tablet, and mobile
- **Custom Typography** - Uses Google Fonts (Rye, Cinzel, Special Elite)
- **No External Dependencies** - Pure HTML, CSS, and vanilla JavaScript
